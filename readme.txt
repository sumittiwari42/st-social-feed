== Description ==
ST social feed is very simple and low weight slide show plugin which helps WordPress/dashboard user to fetch social feed and Show on page by using shortcode i.e. [st_social_feed]


== Installation ==
*	Download the plugin .zip file.
*	Login in to admin Click Plugins -> Add New -> Upload.
*	Find ST social feed in plugin list  and activate it.
*	After activate plugin you will see ST social feed menu at left side on WordPress dashboard.
